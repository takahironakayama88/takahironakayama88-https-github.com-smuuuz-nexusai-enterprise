# sunsun サービス概要

## コンセプト

sunsunは、企業向けマルチAIチャットプラットフォームです。OpenAI・Anthropic・Googleの主要3プロバイダーに対応し、組織単位でAPIキーを一元管理。セキュリティと監査機能を備え、社内のAI活用を安全かつ効率的に推進します。

---

## 主な機能

### マルチAIチャット
- OpenAI (GPT-4, GPT-4 Turbo, GPT-3.5 Turbo)、Anthropic (Claude Sonnet 4.5, Claude Sonnet 4, Claude Opus 4.1)、Google (Gemini 2.5 Pro, Gemini 2.5 Flash) の計8モデルに対応
- リアルタイムストリーミング応答
- テキスト・画像・PDF・Office文書などマルチモーダル入力対応（最大10MB/ファイル）
- スレッド管理による会話履歴の保存・検索

### 3つのAIモード
- **高速モード**: 素早い応答が必要な場面に
- **バランスモード**: 速度と品質のバランスを重視
- **高精度モード**: 複雑な分析や詳細な回答が必要な場面に
- 各モードに任意のAIモデルを割り当て可能

### カスタムAIアシスタント
- 組織独自のAIアシスタントを作成（例: 契約書レビューAI、社内FAQ対応AI）
- カスタムシステムプロンプト、アイコン、カラーテーマで個性化
- 会話スターターによるユーザーガイド機能
- カテゴリ管理による分類（例: 人事部、総務部、営業部）
- 公開範囲設定（全員 / オーナーのみ）

### 組織・ユーザー管理
- マルチテナント対応（組織単位でデータ完全分離）
- ロールベースアクセス制御（OWNER / MEMBER）
- OWNERは全管理機能にアクセス可能、MEMBERはチャット機能のみ

### APIキー管理
- 組織単位でAPIキーを設定・更新・解除
- AES-256-GCM暗号化によるセキュアな保管
- APIキーの接続状態をリアルタイム表示

### 監査・コンプライアンス
- **操作ログ**: ログイン、APIキー変更、モード変更、スレッド・アシスタント・カテゴリ操作を記録。操作種別・ユーザー・期間でフィルタリング可能
- **チャット監査**: 全ユーザーの会話履歴を閲覧・検索。キーワード検索対応
- **AI監査レポート**: AIを活用した監査レポートの自動生成。Word出力対応
- 全操作にIPアドレスとタイムスタンプを記録
- CSV / Wordエクスポートによる外部監査対応

### セキュリティ
- JWT認証（Bearer Token方式）
- APIキーのAES-256-GCM暗号化保存
- 組織間データ完全分離
- 全管理操作の監査証跡

## 技術スタック

| 領域 | 技術 |
|------|------|
| フロントエンド | Next.js 16 / React 19 / TypeScript |
| UIライブラリ | Radix UI / Tailwind CSS 4 |
| AI統合 | Vercel AI SDK v6（ストリーミング対応） |
| データベース | Prisma ORM / SQLite |
| 認証 | JWT（JSON Web Tokens） |
| 暗号化 | AES-256-GCM |

## 想定利用シーン

- 社内のAI利用を一元管理し、コスト・セキュリティを統制したい企業
- 部門ごとに専用AIアシスタントを配備し、業務効率化を図りたい組織
- AI利用状況の監査・コンプライアンス対応が求められる企業
- 複数のAIプロバイダーを比較・使い分けたいチーム
